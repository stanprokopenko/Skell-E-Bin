How to Deform Boxes for Dynamic Drawings

Copy Code

Perspective doesn't have to be static and boring. You can add gesture to your perspective by deforming boxes to make them feel more dynamic. This technique is useful in two main scenarios: drawing organic forms like the human body and drawing inorganic forms like buildings or cars.

Curving

Curving a box can make it feel like it's moving or soft. Imagine drawing the front plane of a box and curving the lines instead of keeping them straight. This method can be applied to organic forms like arms, torsos, and animals.

  • Example: Draw a front plane of a box and curve the lines to connect the corners. This creates a box with a C curve gesture.
  • Application: Use subtle curves for a slight personality or exaggerated curves for a more dynamic effect.

Bloating

Bloating a box involves expanding all sides as if filling it with air. This makes the box feel fuller and more dynamic.

  • Example: Inflate a box so that all sides curve outward.
  • Application: This technique can be used to add volume and personality to various forms.

Imploding

Imploding a box means sucking air out of it, curving the lines inward to create a sharp, rigid form. This can make the form feel dangerous or villainous.

  • Example: Curve the lines inward to create a box that feels sharp and rigid.
  • Application: Use this technique for character design, especially for villains, to give them a more menacing appearance.

Tapering

Tapering a box involves making it thicker or thinner on one side. This can add weight or imbalance to the form.

  • Example: Make a box heavier at the bottom by tapering the lines.
  • Application: Use tapering for body parts like forearms and legs, where the form naturally gets thinner or thicker.

Twisting

Twisting a box is especially useful for drawing torsos. Visualize the top and bottom planes of the box at different angles and connect them with curved lines.

  • Example: Draw the top plane facing one direction and the bottom plane facing another, then connect them.
  • Application: This technique is great for drawing torsos and can be combined with other deformations for more complex forms.

Leaning

Leaning boxes can add personality and motion to your drawings. Tilt the box slightly to give it a dynamic feel.

  • Example: Tilt a chimney on a roof to give the house more character.
  • Application: This technique can be used for both organic and inorganic forms to add a sense of motion and personality.

Combining Deformations

Combining all these methods can create highly dynamic and interesting forms. For example, you can draw a dog house with various deformations to make it look fun and lively.

  • Example: Start with a basic shape and apply curving, bloating, imploding, tapering, and twisting to different parts of the form.
  • Application: Use these techniques to balance perspective rules with creative deformations, enhancing the visual effect.

* * *

Play with shapes and forms to make your drawings more dynamic and fun. By using these deformation techniques, you can transform static boxes into lively, interesting forms. Have fun with it, and see how these methods can bring your drawings to life!
