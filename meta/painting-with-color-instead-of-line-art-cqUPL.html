Painting with Color Instead of Line Art

Copy Code

I'll show you a simple process to create a digital painting starting directly in color. Follow these steps to focus on color and process.

Choosing a Reference

  • Select an easy-to-draw reference to focus on color and process.
  • Example: A photo taken from a moving car with interesting lighting and clouds.

Creating Compositional Thumbnails

  • Start with rough block-ins to find a pleasing arrangement of shapes.
  • Experiment with the ratio of ground to sky to establish a focal point.
  • Refine the chosen sketch by adjusting the crop and arrangement of shapes.

Refining the Sketch

  • Merge all visible layers onto one layer.
  • Copy the sketch to a new high-resolution document.
  • Darken the background layer and reduce the opacity of the thumbnail.
  • Create a new layer for the drawing and establish perspective using tools like Perspective Tools by Sergey Kritsky.

Blocking in Colors

  • Create separate layers for different elements (e.g., ground, sky, house).
  • Lay in basic gradients and block in base colors for various elements.
  • Use temperature variations to add interest and separate planes of space.

Painting the Clouds

  • Keep clouds lighter than the foreground to maintain value separation.
  • Focus on creating nice shapes that interact with the foreground.
  • Add form to the clouds by considering light sources and using warm and cold grays.

Adding the Sunset Glow

  • Use a blending mode to add the bright orange glow from the sun.
  • Experiment with adjustment layers to push and pull colors.

Final Painting and Refining

  • Refine and add details to the painting.
  • Use texture brushes for elements like trees and bushes.
  • Reinforce perspective by varying the scale of textures.
  • Add final details to elements like the house, road, and phone pole.

Post-Processing

  • Add effects like light bloom and color harmonization.
  • Use gradient maps and multiply layers for final adjustments.
  • Review and add any final details to complete the painting.

Assignment

  • Choose a comfortable reference photo, preferably a landscape or still life.
  • Design a composition with thumbnail sketches.
  • Execute the final painting using the learned tools and techniques.
  • Include the reference photo when posting the assignment.
